CSS Box Model Example

Box 1 (box-sizing: content-box)
This box has padding, a red solid border, and a margin. The width only applies to the content area.
Box 2 (box-sizing: border-box)
This box has more padding, a dashed border, and margin. The width includes padding and border.
Box 3 (default box-sizing: content-box)
This box uses default box-sizing, has different padding and a green dotted border.
